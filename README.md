Things you'll need:

* [Git](http://git-scm.com/downloads)

* [Python-twitter](https://github.com/bear/python-twitter)
* [Peewee Python ORM](https://github.com/coleifer/peewee)
* [MySQL Database](http://dev.mysql.com/downloads/mysql/)
